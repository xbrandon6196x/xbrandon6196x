# Portafolio Web | Brand Keymolent

Portafolio personal de Brand Keymolent Olvera, estudiante de Ingeniería de Software y Redes con experiencia en telecomunicaciones, soporte técnico, análisis financiero, atención al cliente y desarrollo Front-End.

El sitio presenta mi perfil profesional, proyectos, habilidades, certificaciones, CV descargable, enlaces de contacto y un juego integrado llamado Monstermon.

## Vista General

- Página estática construida con HTML, CSS y JavaScript vanilla.
- Diseño responsive con sidebar, menú móvil y modo claro/oscuro.
- Soporte bilingüe español/inglés.
- Reproductor de música controlado por el usuario.
- Serpiente ambiental translúcida con objetivo en cruz que recorre la página sin bloquear la lectura.
- Secciones de proyectos, habilidades, certificaciones y contacto.
- Modal interactivo con el juego Monstermon.

## Cómo Ejecutarlo

No requiere instalación ni dependencias.

1. Clona el repositorio:

```bash
git clone https://github.com/xbrandon6196x/xbrandon6196x.git
```

2. Abre `index.html` en tu navegador.

También puedes levantar un servidor local simple:

```bash
python -m http.server 5500
```

Luego visita:

```text
http://localhost:5500
```
